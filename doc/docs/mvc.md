# MVC

由于 Jboot 是基于 JFinal 进行二次开发的，因此 Jboot 的 MVC 相关功能是来至于 JFinal 提供的功能。

JFinal 的相关文档： [https://www.jfinal.com/doc/3-1](https://www.jfinal.com/doc/3-1)


## 目录

- Controller : 控制器
- Action ：请求的基本单位
- Interceptor ： 拦截器
- FixedInterceptor ：永久拦截器
- Handler ： 处理器
- Render ：渲染器
- Session 
- Cookie
- Jwt ： Json Web Token
- FlashMessage
- Validate ： 验证器
- 安全
